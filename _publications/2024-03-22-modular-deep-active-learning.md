MedDeepCycleAL bundles model selection, query strategies, annotator UI, and deployment plumbing into a single modular toolchain so clinicians can iteratively curate OCT datasets. The technical report documents engineering decisions, provides configuration recipes, and illustrates how the framework plugged into Ophthalmo-AI.

Learn more in the [arXiv preprint](https://arxiv.org/abs/2403.15143).
